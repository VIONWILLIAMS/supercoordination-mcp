## ⚡ 接入只需3步（2分钟）

### 步骤1: 创建工作目录
```bash
mkdir -p ~/ClaudeWorkspace/supercoordination-mcp
cd ~/ClaudeWorkspace/supercoordination-mcp
```

### 步骤2: 配置MCP连接
```bash
cat > .mcp.json << 'EOF'
{
  "mcpServers": {
    "supercoordination": {
      "type": "http",
      "url": "http://192.168.1.3:3000/mcp"
    }
  }
}
EOF
```

### 步骤3: 启动Claude Code
```bash
claude
```

### 步骤4: 验证连接（在Claude中输入）
```
列出所有成员
```

如果看到成员列表（包括张三、李四），说明连接成功！🎉
